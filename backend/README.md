# HRMS Backend

This directory contains backend-related scripts and the database schema.

## Setup

1.  Navigate to this directory:
    ```bash
    cd backend
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```

## Scripts

-   **`set_admin.js`**: Promotes a user to admin.
    -   Usage: `node set_admin.js <email>`

## Database Schema

-   **`schema.sql`**: Contains the SQL definitions for the Supabase database.
